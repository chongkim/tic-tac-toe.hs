tic-tac-toe.hs
==============

Tic Tac Toe in Haskell


Install
---

Get the [Haskell Platform](http://www.haskell.org/platform/)


Running
---

On Unix, type:

```bash
runghc Main.hs
```

If you want it as a standalone executable, you can type:

```bash
ghc Main.hs -o tic
```

The executable will be named `tic`
